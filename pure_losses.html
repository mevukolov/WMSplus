Обновить данные
Для обновления данных:
  1. Перейдите по ссылке в Чистые списания .
  2. Поставьте в поле «Дата первого списания» вариант «Прошлый месяц» (или «Last month»). Нажмите «Применить» (или «Apply»).
  3. Укажите название вашего логистического объекта в поле «Наименование лог-объекта».
  4. Нажмите «Применить фильтры» (или «Apply filters»).
  5. Лостризон и Флаг оприходования менять не нужно: система самостоятельно уберет лишнее.
  6. В окне «Детализация списаний по первой дате по лог-объектам для СЦ» нажмите «Скачать» (или «Download»), далее «Экспорт Excel» (или «Export Excel»).
  7. Загрузите полученный файл в это окно.
